# SFX setup (manual download)

В PR `*.wav` файлы удалены намеренно.

Скачать бесплатные звуки можно отсюда:
- Kenney Interface Sounds (ZIP): https://kenney.nl/assets/interface-sounds

После скачивания положите/переименуйте файлы в проект так:
- `assets/sfx/jump.wav`
- `assets/sfx/coin.wav`
- `assets/sfx/victory.wav`
- `assets/sfx/game_over.wav`

> Примечание: игра запускается и без этих файлов (аудио отключится автоматически),
> но для звука нужны файлы по путям выше.
